# Simon Says Game
An interactive memory-testing game built with vanilla JavaScript.

### 🎮 How to Play
* The game flashes a sequence of colors.
* The player must repeat the sequence by clicking the buttons in the correct order.
* With every successful round, the sequence gets longer and faster!

### 🚀 Features
* **Dynamic Leveling:** Difficulty increases as the player progresses.
* **Visual Feedback:** Custom CSS animations for button flashes and user clicks.
* **Game Logic:** Handles sequence generation, user input validation, and "Game Over" states.

### 🛠️ Tech Stack
* **HTML5:** Semantic structure for the game board.
* **CSS3:** Flexbox layout and keyframe animations.
* **JavaScript (ES6):** DOM manipulation, event listeners, and game state management.
